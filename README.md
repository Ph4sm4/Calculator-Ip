# Calculator-Ip
Input an Ip address as well as subnet mask and receive calculated values

### Calculations
- Net address
- Broadcast address
- Number of possible hosts
- First host address
- Last host address

### Settings
You can decide which input you would like to display or hide

### My ip button
Gets your local ip address
